## 关键技术点

### 洗牌算法（Fisher-Yates）
```
从数组末尾向前遍历，每次生成 [0, i] 随机索引并交换
```

### 翻牌动画（CSS 3D）
```css
.card-flip {
  perspective: 1000px;
  transform-style: preserve-3d;
}
.card-flip.flipped {
  transform: rotateY(180deg);
}
```

### 海报生成
```
uni.canvasToTempFilePath → 绘制牌面 + 文字 + 装饰 → 保存/分享
```

### 本地存储策略
```
Key: 'tarot-records' → JSON Array → max 50 items → FIFO
```
